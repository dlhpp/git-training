In git training project:

1. Open folder in VS Code

2. Open index.html

3. View changes in "Pending Changes" view in VS Code

3. From command prompt: >git status
    Do you see any changes?

4. Change format from LF to CRLF (lower right tray) and save file

5. View changes in "Pending Changes" view in VS Code

6. From command prompt: >git diff index.html
    Do you see any changes now?
